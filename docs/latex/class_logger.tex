\doxysubsection{Detailed Description}
A utility class for logging messages at different log levels. 

The {\ttfamily \doxylink{class_logger}{Logger}} class provides methods to log messages at various levels (error, warning, info, debug). It allows setting a global log level to control the verbosity of the logs.
