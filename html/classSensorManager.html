Detailed Description

A utility class for managing the SCD30 and BMP280 sensors.

The SensorManager class handles sensor initialization, calibration, and data retrieval for CO2, temperature, humidity, and pressure readings.

Checks if the SCD30 sensor needs calibration and performs calibration if necessary.

This method checks the EEPROM for a calibration flag. If calibration has not been performed, it calibrates the SCD30 sensor and updates the EEPROM.
